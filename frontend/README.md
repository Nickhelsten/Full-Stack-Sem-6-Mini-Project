# Frontend

React frontend for the Online Event Registration Platform.

## Pages Included

- Home page with event list
- Event details page
- Registration form page
- Admin dashboard
- Create or edit event page
- Registrations list page

## API Connection

The frontend connects to:

`http://localhost:8080/api`
